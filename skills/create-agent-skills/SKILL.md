# This skill has been merged into /carmack

All content from this skill is now built into `/carmack`.

**Use `/carmack` for all engineering work** — it includes:
- Skills are prompts: YAML frontmatter + markdown, no XML tags
- Progressive disclosure: SKILL.md under 500 lines, split into reference files
- Required frontmatter: name (64 chars), description (1024 chars, includes trigger keywords)
- Naming conventions: gerund form (processing-pdfs, reviewing-code)
- Audit checklist for valid, well-structured skills
- Anti-patterns: XML tags, vague descriptions, deep nesting, too many options
